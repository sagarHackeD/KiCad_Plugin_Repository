# KiCad Plugin Repository

Welcome to the KiCad Plugin Repository! This repository serves beta plugins before thy are submitted to the KiCad.

## Getting Started
To get started with using the plugins from this repository, follow these steps:
1. **Add Repository to KiCad PCM:**
   ```bash
   https://raw.githubusercontent.com/sagarHackeD/KiCad_Plugin_Repository/refs/heads/main/repository.json
   ```
